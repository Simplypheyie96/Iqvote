# IQ Vote - Quick Start Guide

## 🚀 For the First Person (Admin)

### 1. Create Your Account
- Open the app → Click **"Sign Up"**
- Enter your name, role, email, password
- You'll automatically become the **admin** (first user gets admin rights!)

### 2. Add Your Team (Admin Tab → Employees)
- Click **"Admin"** in the header → **"Employees"** tab
- Click **"Add Employee"** for each team member
- Enter: Name, Email, Role (and optionally Department, Admin status, Image)

### 3. Create an Election (Admin Tab → Elections)
- Go to **"Elections"** tab → **"Create Election"**
- Set name (e.g., "Employee of the Month - Nov 2024"), start/end dates
- Click **"Create Election"**

### 4. Set Who Can Vote (Admin Tab → Eligibility)
- Go to **"Eligibility"** tab → Click on your election
- Check **"Can Vote"** for voters
- Check **"Can Be Voted For"** for candidates
- Changes save automatically ✅

### 5. Invite Your Team
- Share the app URL with your team
- They sign up using the **same email** you added
- They're automatically linked to their employee profile

## 👥 For Employees

### Sign Up
- Click **"Sign Up"**
- Use the **same email** your admin added for you
- Create your password

### Vote
1. Go to **"Vote"** tab
2. Select 3 different people (1st, 2nd, 3rd choice)
3. Optionally write reasons (anonymous!)
4. Click **"Submit Vote"**

**Rules**: Vote once per election. Cannot change after submitting.

### View Results
- **"Leaderboard"** tab → Filter by year/month/all-time
- See points, vote breakdown, and anonymous messages
- **"My History"** tab → See your past votes

## 🔑 Key Points

### Admin Rights
- ✅ **First user** = automatic admin
- ✅ **Admins can make others admins** (Edit employee → Check "Is Admin")
- ✅ **Only admins see the Admin tab** (with shield icon 🛡️)

### Voting System
- 1st choice = **5 points**
- 2nd choice = **3 points**  
- 3rd choice = **2 points**
- Vote once per election (no changes allowed)
- Optional anonymous voting reasons

### Election Eligibility
- ⚠️ **IMPORTANT**: Set eligibility for every election!
- Without eligibility, elections won't work properly
- Control who votes AND who can be voted for
- Can be different for each election

## 🎯 Common Tasks

### Make Someone an Admin
1. Admin → Employees
2. Click edit icon on their card
3. Check "Is Admin"
4. Update Employee

### Start a New Monthly Election
1. Admin → Elections → Create Election
2. Name it "Employee of the Month - [Month Year]"
3. Set start/end dates for the month
4. Admin → Eligibility → Set voters and candidates
5. Done! ✅

### Add a New Employee Mid-Month
1. Admin → Employees → Add Employee
2. Admin → Eligibility → Find current election
3. Check "Can Vote" and/or "Can Be Voted For"
4. Tell them to sign up with their email

### View Historical Results
1. Leaderboard tab
2. Use year/month dropdowns
3. Select specific time period
4. View aggregated results

## ⚠️ Common Mistakes to Avoid

❌ **Forgetting to set eligibility** → Election won't work  
✅ Always go to Eligibility tab after creating election

❌ **Adding employees with wrong email** → They can't sign up correctly  
✅ Use their actual work email

❌ **Not uploading employee photos** → Generic avatars everywhere  
✅ Upload photos for better UX

❌ **Setting wrong election dates** → Can't edit after creation  
✅ Double-check dates before creating

❌ **Employees sign up with different email** → Won't link to profile  
✅ Tell them to use exact email admin entered

## 📊 Understanding the Leaderboard

### Top 3 Podium
- 🥇 Gold medal for 1st place
- 🥈 Silver medal for 2nd place  
- 🥉 Bronze medal for 3rd place

### Stats Shown
- **Total Points**: Sum of all votes
- **Vote Breakdown**: Count of 1st/2nd/3rd place votes
- **Messages**: Click icon to read anonymous reasons

### Time Filters
- **All Time**: Everything since the beginning
- **Year View**: All elections in selected year
- **Month View**: Elections in specific month of year
